NULLEXISTENCE

Where Is Everything – Steganography Challenge

Author: Arveen (Neevra_Llig)

Category: Steganography

Objective: Uncover a hidden flag concealed across three seemingly empty files through metadata analysis, steganography extraction, and password recovery.

Abstract

Presented with an HTTP 404 "Everything Not Found" and three files that appear blank or trivial, participants are challenged to question surface-level assumptions. What appears empty may conceal hidden structure, metadata, or unconventional data storage. This challenge tests attention to detail, curiosity, and the ability to uncover what is deliberately obscured beneath the illusion of nothingness.

1. Files Given

empty.png
empty.txt
empty.js

2. Inspecting empty.js

Opening empty.js revealed two distinct parts. The first was a block of fake legal instructions designed to mislead — a red herring. The second was actual JavaScript: 

const DRIFT_LOG = {
  title: "DEEP SKY SURVEY - DRIFT LOG",
  lines: [
    "Space looks empty until you measure what you can't see.",
    "Some signals don't live in letters... they live between them.",
    "If this file feels blank, that's the point.",
    "We logged the anomaly as: \"the void is carrying data\".",
  ],
};

const TRANSMISSION_SCRAP = "aHR0cHM6Ly9sZWVrc3Bpbi5jbw==";

Decoding the Base64 string in TRANSMISSION_SCRAP yielded either a troll or a thematic distraction — it does not directly reveal the flag.

3. Inspecting empty.png

Running zsteg against empty.png uncovered hidden data in the image metadata: 

$ zsteg empty.png

meta Comment .. text:
"m2v9kq1/ax0n7uH3p9t5z1aA8w0e6yHk4n2u7sA1v3fJ9c0r5dU2b7m4sS6t1qT8x2n7pT0r6l
1k+R9a4m7uO3p8c2yL5v0n4hL2j6s9dI7q1w3eN8z0p5uG=="

This led to the extraction of a file: payload.zip. Inside was a password-protected flag.png.

4. Cracking the Zip Password

With help from teammates NxStrip and 0xTak, the zip password was recovered. NxStrip pointed to aperisolve.com where, after extracting flag.png, a string section at the bottom of the image contained the flag. 

$ python3 decode.py
Whisper recovered: ZIP_PASSWORD=D4rKm47T3rrr;END[...]

5. Extracting the Flag

With the zip unlocked, running strings and zsteg against flag.png revealed the flag appended as extra data after the image end chunk (IEND): 

$ strings flag.png
IHDR
IDATx
r=Oo
'lrY
IEND
UVT{N0th1nG_iS_3mp7y_1n_sP4c3}

$ zsteg -a flag.png
[?] 32 bytes of extra data after image end (IEND), offset = 0x2008
extradata[0]  .. text: "\r\nUVT{N0th1nG_iS_3mp7y_1n_sP4c3}"

Big thanks to NxStrip and 0xTak for their help finding the password.
